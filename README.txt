1 - Instalar uma versão do JDK, caso não o tenha. Link para instalar: 
    https://www.oracle.com/java/technologies/downloads/
2 - Instalar a diretoria "apache-tomcat-10.1.25"
3 - Instalar a diretoria da base de dados
4 - Colocar a base de dados na diretoria do Utilizador, exemplo:
    C:\Users\*Nome_do_Utilizador*\
5 - Alterar o path do "startTomcat" e do "stopTomcat" para o path onde instalou o jdk
6 - Correr o "startTomcat"
7 - Colocar no browser: https://localhost:8443/projeto_gslab/

Notas :
Para ver codigo ou importar para o eclipse use o .war ou o zip
No ficheiro Users tem a informacao utilizadores que existem na BD
